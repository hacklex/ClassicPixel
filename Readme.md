# Simple Pixel Art Image Editor

![Logo](/Assets/Screenshot.png)

This is a simple pixel art image editor built almost entirely by AI :)

Works on macOS too!

## Features

- Rectangular selection
- Magic wand selection
- Pencil, eraser, and fill tools
- Color picker
- Palettes
